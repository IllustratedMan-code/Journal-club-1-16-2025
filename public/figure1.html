"(TF) activity is a latent variable pooling the effects of post-transcriptional and post-translational modifications."
Let's break this down. TF activity is a "latent" variable, meaning that it is not directly observable. This definition is a little obtuse. It essentially means "whatever the protein is doing after it is made". Because TF activity is not directly observable, the activity must be estimated using other variables that are accessible. Namely, known interactions using the Subtiwiki and the transcriptional data.
